Install node via Node Version Manager
$ curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.32.1/install.sh | bash

This file gets run on every log in, but you can initilize it manually to get access to nvm ls
source ~/.bashrc

Install node with specified version
nvm install X.X.X

Check if installed
node --version

To exit:
ctrl + z
& run instance in the background:
bg %1

Install nginx
sudo apt-get install nginx

Open nginx config
cat /etc/nginx/sites-available/default

List jobs running in the background
jobs

Search for file
find / -type f -name "index.js"

Kill all node processes
killall -9 node

To keep processes running
npm i -g pm2
pm2 start index.js

Restart PM2 on server restart
pm2 startup

Save current processes so they run when PM2 restarts
pm2 save

To list all processes use
pm2 ls
