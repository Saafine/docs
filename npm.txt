Globally installed npm packages
npm install -g npm-check-updates

Save exact version
npm install moment --save --save-exact

^match minor version 1.X.0
~match patch versrion 1.0.X

Get outdated packages
npm outdated

Display what's installed globally
npm list -g --depth=0

Display whats installed in node_modules
npm list --depth=0 

Remove extraneous packages
npm prune --production=false

Show any new dependencies for the project in the current directory:
ncu

Build a tree of dependendecies
npm shrinkwrap --dev

To make sure your modules are up to date and locked
npm install npmvet -g
npmvet -r inlinetable

To update a package:
1. npm outdated
2. npm install --save-XXX --save-exact
3. remove npm-shrinkwrap.json
4. npm shrinkwrap --dev

To update all packages:
1. ncu
2. ncu -u // updates package.json
3. remove npm-shrinkwrap.json
4. npm install
5. npm shrinkwrap --dev
6. npm list --depth=0  // check for extraneous packages
